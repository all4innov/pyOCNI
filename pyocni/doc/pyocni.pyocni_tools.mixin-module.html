mixIn(base, addition)

 

Mixes in place, i.e. the base class is modified. Tags the class with a list of names of mixed members.

unMix(cla)

 

Undoes the effect of a mixin on a class. Removes all attributes that were mixed in -- so even if they have been redefined, they will be removed.
